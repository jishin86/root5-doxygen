class TSPHE: public TShape

 

 SPHE is a Sphere. It has 9 parameters:

     - name       name of the shape
     - title      shape's title
     - material  (see TMaterial)
     - rmin       minimum radius
     - rmax       maximum radius
     - themin     theta min
     - themax     theta max
     - phimin     phi min
     - phimax     phi max
