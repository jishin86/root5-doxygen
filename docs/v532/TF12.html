class TF12: public TF1


 a projection of a TF2 along X or Y
 has the same behaviour as a TF1

      Example of a function

   TF2 *f2 = new TF2("f2","sin(x)*sin(y)/(x*y)",0,5,0,5);
   TF12 *f12 = new TF12("f12",f2,0.1,"y");
   f12->Draw();
